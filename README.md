# Suites Mine Hotel Marketing CRM

Preview front-end d'un CRM marketing premium pour Suites Mine.

Le dashboard est pense pour l'hotellerie boutique et l'aparthotel :

- planification marketing
- suivi invites et comptes corporate
- newsletter et segmentation
- calendrier editorial
- automatisations hotelieres
- reputation et reviews
- reporting executif

## Apercu local

Ouvrir `index.html` directement dans un navigateur.

## Fichier principal

- `index.html` : preview statique complete, front-end only.
